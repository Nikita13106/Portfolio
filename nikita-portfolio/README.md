# Nikita Mishra - Developer Portfolio

A dark-themed **systems-style developer portfolio** built using modern web technologies.
Designed to feel like a developer dashboard, it showcases projects, skills, and experience with smooth animations and a clean UI.

---

## 🚀 Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion

---

## 🛠️ Run Locally

```bash
npm install
npm run dev
```

Open the local server URL 

---

## 📦 Build for Production

```bash
npm run build
npm run preview
```

---

## ✏️ Editing Content

All portfolio content including:

* Projects
* Skills
* Experience
* Achievements
* Links

is managed from a single file:

```
src/data/content.js
```

Update this file to instantly reflect changes across the entire site.

---

## 📌 Features

* Fully responsive design
* Smooth animations using Framer Motion
* Optimized for performance and readability
* Built from scratch - not a template
